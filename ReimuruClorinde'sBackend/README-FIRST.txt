UPLOAD THESE FILES TO THE ROOT OF YOUR GITHUB REPO.

Final GitHub structure should be:

server.js
package.json
.env.example
.gitignore
README-FIRST.txt
FullWebsiteCode/
  index.html
  subjects.html
  settings.html
  documents.html
  flashcards.html
  quiz.html
  style.css
  script.js
  Logo.png

Important:
- Do NOT create an extra AppCode folder.
- Do NOT upload old duplicate css/ or js/ folders.
- Keep all frontend files inside FullWebsiteCode.
- Upload the CONTENTS, not a parent wrapper folder.

Railway variables to set later:
- OPENAI_API_KEY
- OPENAI_MODEL (optional, example: gpt-4.1-mini)

If OPENAI_API_KEY is not set, the app still runs using fallback logic.
